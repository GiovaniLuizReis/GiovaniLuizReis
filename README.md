# Giovani Reis

**`Desenvolvedor FullStack`**

Mestrando em Sistemas de Informação pela Universidade de São Paulo (EACH - USP). Graduado em Análise e Desenvolvimento de Sistemas pelo Instituto Federal de São Paulo (IFSP), Técnico em Eletrônica pela mesma instituição e Técnico em Informática pelo IFSULDEMINAS. Atualmente, é bolsista de pesquisa no Centro de Tecnologia da Informação Renato Archer (CTI), atuando no projeto IARA-Saúde. No âmbito deste projeto, é responsável pelo desenvolvimento de aplicações mobile, sistemas web e modelagem de bancos de dados, integrando soluções de software para a área da saúde. Colabora como pesquisador voluntário no Instituto Federal de Mato Grosso do Sul (IFMS) no projeto ''Detecção de Estresse Agrícola com Imagens de Satélite e Inteligência Artificial'' e possui experiência como Desenvolvedor Full Stack (Java, Django e JavaScript). Apresenta sólida produção técnica em Ciência de Dados e Visão Computacional, com foco no desenvolvimento de modelos preditivos para diagnósticos médicos e monitoramento agrícola de precisão, utilizando Python, Scikit-Learn e TensorFlow.
